# Online Examination & Evaluation Database

## Description
This project is a DBMS-based design for managing online exams.

## Files Included
- schema.sql → Tables
- data.sql → Sample data
- queries.sql → SQL queries
- views.sql → Views
- triggers.sql → Triggers
- procedure.sql → Stored Procedures
- index.sql → Indexing
- transactions.sql → Transactions

## How to Run

1. Open MariaDB
2. Run files in order:

SOURCE schema.sql;
SOURCE data.sql;
SOURCE views.sql;
SOURCE triggers.sql;
SOURCE procedure.sql;
SOURCE index.sql;
SOURCE transactions.sql;
SOURCE queries.sql;

## Features
- Normalized schema
- Constraints & relationships
- Stored procedures
- Triggers
- Transactions
